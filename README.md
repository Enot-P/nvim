# Neovim Configuration

Конфигурация Neovim с использованием lazy.nvim для управления плагинами.

## Плагины

| Плагин | Описание |

| **lazy.nvim** | Менеджер плагинов для Neovim, обеспечивающий ленивую загрузку и оптимизацию производительности. Автоматически управляет установкой, обновлением и удалением плагинов. Поддерживает параллельную загрузку и отслеживание изменений конфигурации. |

| **folke/which-key.nvim** | Плагин для отображения доступных хоткеев при нажатии префикса. Помогает запомнить комбинации клавиш и показывает подсказки для всех доступных команд. Автоматически группирует хоткеи по категориям и отображает их в удобном формате. |

| Хоткей | Режим | Описание |
|--------|-------|----------|
| `<Space>` | - | Leader клавиша (используется как префикс для большинства команд) |

## Структура конфигурации

```
nvim/
├── init.lua                 # Точка входа, загружает опции, хоткеи и lazy
├── lazy-lock.json           # Закреплённые версии плагинов
└── lua/
    ├── lazy_setup.lua       # Настройка lazy.nvim
    ├── config/
    │   ├── options.lua      # Базовые опции Neovim
    │   └── keybinds.lua     # Глобальные хоткеи
    └── plugins/
        ├── init.lua         # Импорт всех плагинов
        ├── which-key.lua    # Конфигурация which-key
        ├── snacs.lua        # Конфигурация snacks.nvim
        ├── theme/
        │   ├── tokyonight.lua
        │   └── lualine.lua
        └── lsp/
            ├── lsp.lua              # Базовый LSP
            ├── mason.lua            # Mason и установка инструментов
            ├── treesitter.lua       # Treesitter
            ├── formating.lua        # Conform (форматтер)
            ├── linting.lua          # nvim-lint
            ├── dap.lua              # Debugger (DAP)
            ├── flutter.lua          # Flutter tools
            └── nvim-cpm.lua         # Автодополнение (nvim-cmp)
```
